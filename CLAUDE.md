# Project Overview
This is a Neovim configuration with LSP, autocomplete, and custom keymaps.

Read all the files into your context.

  ## Key files:
  - `init.lua` - Main config entry point
  - `lua/theprimeagen/lazy/lsp.lua` - LSP and autocomplete setup
  - `lua/theprimeagen/remap.lua` - Custom keybindings
